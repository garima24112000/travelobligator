# TravelObligator

TravelObligator is an AI-powered travel planning platform that generates personalized, route-aware, and editable itineraries.

## Problem

Travel planning today is:
- fragmented across multiple websites
- generic and not personalized
- often unrealistic in terms of route and time
- difficult to modify once created

## Solution

TravelObligator:
- collects detailed user preferences
- generates a structured itinerary using AI + travel data
- shows an interactive dashboard + map
- recommends accommodations and transport
- allows iterative refinement of the itinerary

## Key Features

- Personalized itinerary generation
- Day-wise structured plans
- Map-based route visualization
- Accommodation recommendations
- Transport recommendations
- Budget estimation
- Feedback-based regeneration

## Tech Stack (Planned)

Frontend: Next.js, React, TypeScript, Tailwind  
Backend: FastAPI or Node  
Database: PostgreSQL  
AI: Structured LLM outputs  
Maps: Mapbox / Google Maps
